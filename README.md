<h2 align="center">Finance FastAPI</h2>


### Описание проекта:
Веб-приложение финансов написанный на FastAPI.
- JWT авторизация
- CRUD пользователей
- CRUD категорий
- CRUD транзакций
- CRUD баланса
- Отправка Email

### Инструменты разработки

**Стек:**
- Python >= 3.11
- FastAPI == 0.96.0
- PostgreSQL
- Docker
- Redis
